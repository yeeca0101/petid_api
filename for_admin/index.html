Folder Seed Upload

설명

폴더명 기준으로 여러 펫의 seed 이미지를 한 번에 등록합니다.

폴더 구조는 root/pet_name/image.jpg 형태를 권장합니다.

Exemplar는 query/reference 이미지셋이므로 각 이미지에 단일 개체가 선명하게 보이는 구성을 권장합니다.

기존 pet에 추가는 같은 이름이 있으면 그 pet에 이어 붙이고, 새 pet 생성은 suffix를 붙여 새 pet으로 만듭니다. 중복 이름이면 등록 중단은 충돌 시 실패 처리합니다.

예시:

root/
  뽀미/
    xxx.jpg
    xxx.png
  초코/
    xxx.jpg
    xxx.jpeg
폴더 구조를 읽어 pet별 seed exemplar를 일괄 등록
중복 이름 처리
대기 중 폴더 내부 파일을 순차 등록합니다.

설명

Daily는 단일/복수 개체 이미지 모두 허용합니다.

업로드 후에는 '전체' 또는 '미분류' 탭에서 바로 확인할 수 있습니다.

Daily Ops

설명

현재 날짜의 ACCEPT 결과를 pet별 버킷으로 고정하고 manifest를 저장합니다.

 설명

가장 최근에 확정한 버킷 기준으로 ZIP 파일을 내려받습니다.

 설명

Daily ZIP은 해당 날짜 폴더 아래 이미지와 짝이 되는 _anno.json을 함께 제공합니다.

Annotation에는 이미지 내 각 개체의 name, pet_id, bbox, assignment_status가 들어갑니다.

모두 다운로드 또는 적어도 하나 ACCEPTED가 있는 이미지만 골라 받을 수 있습니다.
